Acerca del Inventario

Este subsistema está diseñado para facilitar la gestión del inventario de cerdos, proporcionando al dueño una serie de funcionalidades que permiten una administración eficiente y efectiva de su granja.

Usuario del Subsistema

  • Dueño

Requerimientos por Usuario

Requerimientos del Dueño

El dueño debe ser capaz de:

  1. Visualizar el Inventario Actual: Ver una lista completa de todos los cerdos en el inventario, con información sobre:
    • ID de cerdo
    • Raza
    • Peso
    • Fecha de nacimiento
    • Etapa de crecimiento
  2. Agregar Nuevos Cerdos: Incluir nuevos cerdos con los siguientes datos:
    • ID de cerdo
    • Raza
    • Peso
    • Fecha de nacimiento
    • Etapa de crecimiento
  3. Editar Cerdos Existentes: Modificar la información de cerdos ya registrados.
    • ID de cerdo
    • Raza
    • Peso
    • Fecha de nacimiento
    • Etapa de crecimiento
  4. Eliminar Cerdos: Eliminar cerdos con confirmación para evitar errores.
  5. Generar Reportes: Crear informes por:
    • Periodo de tiempo
    • Tamaño
    • Raza
  6. Registrar Movimientos del Inventario: Registrar entradas y salidas, incluyendo:
    • Fecha y hora
    • Tipo de operación
    • Cantidad afectada
