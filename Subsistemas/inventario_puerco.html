Invetario de Puercos

Este subsistema está diseñado para facilitar la gestión del inventario de cerdos, proporcionando al dueño una serie de funcionalidades que permiten una administración eficiente y efectiva de su granja.

Usuario del subsistema

  • Dueño

Requerimientos por usuario

Funcionalidades de que debe ser capaz el dueño:

  • Visualizar el Inventario Actual
    • ID de cerdo
    • Raza
    • Peso
    • Fecha de nacimiento
    • Etapa de crecimiento
  • Agregar Nuevos Cerdos
    • ID de cerdo
    • Raza
    • Peso
    • Fecha de nacimiento
    • Etapa de crecimiento
  • Modificar Cerdos Existentes
    • Raza
    • Etapa de crecimiento
    • Peso
  • Eliminar Cerdos
    • Opción para eliminar cerdos del inventario, con confirmación para evitar errores.
  • Generar Reportes de Inventario
    • Informes sobre:
    • Productos más vendidos
    • Tendencias de ventas
  • Historial de Movimientos de Inventario
    • Registro de todas las entradas y salidas, incluyendo:
    • Fecha y hora
    • Tipo de operación
    • Cantidad afectada
